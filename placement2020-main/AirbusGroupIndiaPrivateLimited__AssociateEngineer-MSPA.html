Physical System Architecture – Associate Engineer

Job Code: Physical System Architecture – Associated Engineer (PSA)

Job Description:

As a Physical System Architecture – Associated Engineer, you will be responsible for a host of activities in the area of Aircraft safety analysis and associated System engineering tasks. Your contribution will span through the entire aircraft life cycle starting from early development stages to delivery of aircrafts.

You will get an opportunity to work on different aircraft programs involving different Aircraft systems

The job-holder is responsible for delivering Global Architecture Mockup and associated System engineering tasks for different aircraft programs on Time, Cost and Quality.

The various activities to be performed are:

  • Creation and update of Physical System architecture based upon a given Target configuration and associated input:
  • Screening and Implementation of Change Request
  • Convergence check of Geometry between Global Architecture Mockup  and  Detailed design (Digital Mockup) and necessary implementation
  • Convergence check of electrical routes/function between Global Architecture Mockup  and CIRCE (Treatment estimation tool)
  • Check and implementation of  Electrical Schematics (Functional diagram) in Global Architecture Mockup
  • Verification and Validation:
  • Analysis of System installation/safety requirement (SIRD) using Global Architecture Mock up
  • Creating Proof of compliance to capture documentary evidence of the analysis
  • Trade off studies:
  • Feasibility studies for new change proposal
  • Providing System Architecture/Installation input for various improvement studies
  • Creation of post mod architecture
  • Check of compliance status of Post Mod against Particular Risks other SIRD requirement
  • Weight/cost/ performance assessment
  • Request for Support Activities:
  • From System design/System Installation
  • For improvement of Process, Methods and Tools
  • Participation to Plateaus
  • Interfacing with relevant counterparts in Europe to ensure maximum effectiveness of Physical Systems Architecture design
  • Project Management
