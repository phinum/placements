JoB Description

We are looking for an entry level Analyst - IT to build and implement functional programs. He/she will work in a team comprising of developers, business analysts, quality control team along with the project manager throughout the life cycle of the project. In this role, he/she should be a team player with a keen eye for detail and excellent programming and problem-solving skills. Your goal will be to learn and implement software concepts, gain expertise on tools & technologies and build applications and systems that solve problems for our clients.

What is expected from me on a day-to-day basis if I join Merilytics?

  • Work with technology and business analysts to design and develop programs in various languages
  • Produce clean, efficient code by following best coding practices and standards
  • Attention to detail, adherence to deadlines and accuracy of the output – some of the core qualities expected from this candidate
  • Ability to create clear and crisp technical documentation during code development
  • Ability to work on feedback and ensure they are never repeated

What are the pre-requisites and skillsets required to apply for this role?

  • Basic familiarity with OOPs & OOAD concepts
  • Knowledge of coding languages (C, C++, Java, Python, SQL etc.) will be preferred
  • Ability to learn new languages, tools and technologies on the job
  • Excellent communication skills – with peers and manager
  • Self-starter with ability to work independently and manage own time well
