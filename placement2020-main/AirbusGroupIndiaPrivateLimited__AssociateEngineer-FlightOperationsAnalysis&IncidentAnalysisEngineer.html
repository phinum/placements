Description :

Associate Engineer - Flight Operations Analysis Engineer & Incident Analysis

Activities:

  • Flight Operations Analysis: Statistical analysis on large set of flight operations data originating from a large number of flights. This includes:
    • Decode and validate data from digital flight data recorders and direct access recorders.
    • Develop algorithms to sample, extract, transform and analyze data.
    • Perform statistical analysis with the help of experts and specialists.
    • Present results on the findings of the analysis to experts and specialists.
  • Incident Analysis (Lite Analysis): Incident analysis performed on a specific flight during acceptance or in-service activity. This includes:
    • Decode and validate data from digital flight data recorders and direct access recorders.
    • Analyze data for incidents
    • Write reports of the findings and send it to the requestors.
  • Algorithm development for event detection: Develop algorithms to detect safety related events from data obtained from flight data recorders.

Department:

Aerospace Engineering/Mechanical Engineering/Systems Engineering.

Required Technical Expertise:

  • Aircraft domain (Flight operations, Flight physics/mechanics, Systems)
  • Analytical skills
  • Basics algorithms and programming (Any language C, C++,  C#, Python etc)

Required Soft Skills:

  • Good presentation & communication skills
