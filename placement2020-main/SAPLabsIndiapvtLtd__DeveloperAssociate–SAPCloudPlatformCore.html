Description :

SAP, Originally known for leadership in enterprise resource planning (ERP) software, SAP has
evolved to become a market leader in end-to-end enterprise application software, database,
analytics, intelligent technologies, and experience management. A top cloud company with 200
million users worldwide, SAP helps businesses of all sizes and in all industries to operate
profitably, adapt continuously, and achieve their purpose.
If Systems Engineering [Virtualization, Containers, Linux, Core Networking, Cloud Native
Development, PaaS, IaaS, Distributed Systems] is your area of interest, then you are applying
at the right place
SAP Cloud Platform is an Open, Industry Standard Platform-as-a-Service offering from
SAP. Built on Cloud Foundry and Kubernetes, it is a platform of choice to extend, integrate
and build innovative cloud native applications at SAP
SAP Cloud Platform Core unit provides the multi-cloud foundation for SAP Cloud Platform
and a layer of abstraction for all applications to run on both public clouds [AWS, Azure, GCP,
AliCloud] and private cloud. Technical Services around data management, platform services,
etc. forms the key portfolio of the unit
Expectations
As a developer at the SAP Cloud Platform Core Unit you will be working on Data Management
Services – provides multi-cloud managed hyperscale backing services like Object Store,
Database & Caching as a Service. On private cloud, we provide Kubernetes native backing
services based on the concept of operators.
Technology Stack:
• Languages – Java, NodeJS, Go, Python, Shell
• Databases – S3, PostgreSQL, MongoDB, Redis
• Container – Docker, K8S. OS - Linux
• IaaS – AWS, Azure, GCP, AliCloud, Private Cloud
Education & Skill Requirements
• Demonstrate excellent quality, design, development & delivery skills
• Sound understanding of cloud native architecture & development
• Masters or bachelor’s Computer Science degree in Engineering with excellent academic
records
